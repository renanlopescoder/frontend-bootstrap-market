# front_end-framework-bootstrap-market

Projeto de curso Alura desenvolvido com a framework css bootstrap e também para as animações o famoso wow.js, este projeto foi desenvolvido como simples layout.

##Tecnologias

 - jQuery framework JavaScript (https://jquery.com/)
 - Bootstrap Framework CSS (http://getbootstrap.com/css/)
 - wow.js uso de animações (http://mynameismatthieu.com/WOW/docs.html)

##Recursos do Projeto

 - Banner animado mostrando imagens onde podemos inserir promoções e produtos mais vendidos chamando a atenção do cliente.
 - Pagina principal mostrando os produtos da loja com animação na rolagem.
 - Página do produto com as informações dos produtos em abas facilitando a visualização e navegação nas informações.
 - Página de carrinho com formulário completo de cadastro do cliente, usando ícones para melhorar a visualização.

Alura course project developed with the CSS framework Bootstrap and also for the famous wow.js animations , this project was developed as a simple layout.
